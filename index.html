📘 React Day 1 – Task: Profile Card using Props

🔍 Learning Objectives

  • External CSS ka sahi use React me
  • Local image ko component me import karna
  • JSX me common HTML tags aur attributes ka use

📚 Concepts to Review

🔹 1. External CSS in React

Har React component ke saath ek separate CSS file ho sakti hai:

import './Profile.css';

React me class ki jagah className use hota hai:

<div className="container">Content</div>

🔹 2. Local Image Import in React

Local image ko import karne ka tarika:

import myImage from '../assets/myphoto.jpg';

Uske baad use JSX me lagate hain:

<img src={myImage} alt="Profile Photo" />

🔹 3. Common JSX Tags and Attributes

  • <img src="..." alt="..." /> – Image
  • <a href="..." target="_blank" rel="noopener noreferrer">Visit</a> – Link
  • className="..." – CSS class
✅ TASK:

Ek reusable React component banao jiska naam Profile.js ho. Yeh component props ke zariye image, name, description aur link receive karega.

🛠️ Task Requirements

  • Component name: Profile.js
  • Props: image, name, description, link
  • Image: Local image import karein aur props se bhejein
  • CSS file: Profile.css ho aur same folder me ho
  • Card ke andar: Image, Name, Description, Website Link ho
  • Use proper JSX: className, alt, href, etc.
📁 Recommended Folder Structure:

/src
├── App.js
├── /components
│     ├── Profile.js
│     └── Profile.css
├── /assets
│     └── profile1.jpg
└── App.css

📌 Additional Instructions

  • App.js ke andar multiple Profile components render karein
  • Har Profile me alag-alag data pass karein (images, names, descriptions, links)
  • Hardcoded content na likhein, sab props se aaye
✨ Bonus (Optional):
  • Hover par link ka color change ho
  • Image circular ho: border-radius: 50%
  • Card center aligned aur responsive ho
  • Hover transitions (button/image par)

🎓 Learning Outcome

Is task ke baad students reusable components banana, props use karna, aur component-based CSS manage karna seekh jaayenge. Ye React ke core skills hain.
